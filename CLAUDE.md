# プロジェクト名
Emotion-Capable Brain-Inspired AI — 情動神経回路の簡略モデル探索

# ミッション

本プロジェクトの目的は、**ヒトの情動に関わる主要脳領域の相互作用を簡略化したモデルで模倣し、情動的な内部状態と行動の生成メカニズムを探索する**ことです。

「抽象的な感情ラベルや状態変数で近似する」のではなく、
**主要な脳領域・神経伝達物質・回路結合・ホメオスタシスの定性的な特徴をモデル化し、それらの相互作用から情動に関連する内部ダイナミクスが生まれる**アーキテクチャを目指します。

# 現時点のモデルの位置づけ

- **スケール**: 547スパイキングニューロン + 4 mean-field領域(各3状態変数)。実際の扁桃体は約1300万ニューロン
- **忠実度**: 回路トポロジーの定性的模倣。パラメータは主に文献推定値+手動設定
- **検証状態**: 定性的な方向性テスト+介入実験。実験データとの定量比較は今後の最優先課題
- **創発性**: 情動readoutは手動設計の関数で導出。回路ダイナミクスからの自動的な情動カテゴリ形成は今後の課題

# アーキテクチャ方針

3つのアーキテクチャが並存するが、位置づけは以下の通り:
- **EmotionBrain** (brian2_circuits/integrated_brain.py): 正式な最終システム。全新規開発はここに統合
- **NeuroPipeline** (neurocircuit/neuro_pipeline.py): WilsonCowan/Brian2/Hybridバックエンド切替用。EmotionBrainは別系統
- **EmotionPipeline** (pipeline.py): レガシー。API互換のために残すが新規開発しない
- **numpy spiking** (circuits/ + spiking/): DEPRECATED。Brian2版に置換済み

# 設計哲学: 3つのモデリング対象レイヤー（目標）

研究レビューに基づき、本プロジェクトは以下の3層を区別します。

1. **回路トポロジー（Circuit Topology）** — 目標忠実度
   - 脳領域（扁桃体、PFC、島皮質、ACC、海馬、視床下部、PAG、線条体、脳幹核）
   - 領域間の解剖学的結合（コネクトーム準拠）
   - 神経質量モデルによる集団活動（興奮性/抑制性バランス）

2. **神経化学モデリング** — 8系統の簡略ダイナミクス
   - 8系統の神経伝達物質ダイナミクス
   - ドーパミン: 報酬予測誤差、動機づけ（VTA→NAc）
   - セロトニン: 気分調整、衝動抑制（縫線核→広域）
   - ノルエピネフリン: 覚醒、注意、闘争逃走（青斑核→広域）
   - GABA: 抑制性制御
   - グルタミン酸: 興奮性伝達
   - オキシトシン: 社会的結合、信頼
   - コルチゾール: HPA軸ストレス応答
   - エンドルフィン: 疼痛調節、快感

3. **身体性モデリング** — HPA軸+自律神経系+内受容の簡略モデル
   - 内受容（interoception）: 内臓・自律神経系からの信号
   - ホメオスタシス: HPA軸、自律神経系の恒常性維持
   - 身体状態が情動に影響する双方向ループ

# 設計原則

- **情動は単一部位の産物ではなく分散ネットワーク統合**として扱う（メタ分析が支持）
- 事実・仮説・設計上の便宜を必ず分ける
- 感情認識、感情生成、感情制御、感情に影響される意思決定、意識の議論を混同しない
- 「会話が感情的に見える」と「情動回路を持つ」は別要件として扱う
- すべての大きな提案には評価計画と失敗モードを付ける
- 評価は3レベル: 神経活動一致 / 回路機能 / 行動再現

# 脳領域アーキテクチャ

以下の脳領域をニューラルマスモデルとして実装し、解剖学的結合で接続する。

1. **扁桃体（Amygdala）**
   - 脅威検出、情動学習、条件付け
   - 入力: 感覚皮質、海馬、PFC
   - 出力: 視床下部、PAG、PFC、線条体
   - 神経修飾: NE↑で感度上昇、5-HT・GABAで抑制

2. **前頭前皮質（PFC）**
   - vmPFC: 価値表象、情動制御
   - dlPFC: 認知制御、作業記憶
   - OFC: 報酬評価、期待違反
   - 入力: 扁桃体、海馬、島皮質、線条体
   - 出力: 扁桃体（トップダウン抑制）、線条体、ACC

3. **島皮質（Insula）**
   - 内受容の統合、主観的感情の基盤
   - 入力: 身体内状態（内受容信号）、扁桃体
   - 出力: ACC、PFC、扁桃体

4. **前帯状皮質（ACC）**
   - コンフリクト監視、エラー検出、痛み処理
   - 入力: PFC、扁桃体、島皮質
   - 出力: PFC（制御要求）、PAG

5. **海馬（Hippocampus）**
   - 文脈記憶、情動記憶のエンコード/検索
   - 入力: 皮質、扁桃体（情動タグ付け）
   - 出力: PFC、扁桃体
   - コルチゾール高値で記憶固定化が変調

6. **視床下部（Hypothalamus）**
   - 自律神経制御、HPA軸の起点、ホメオスタシス
   - 入力: 扁桃体、脳幹
   - 出力: 下垂体（CRH→ACTH→コルチゾール）、自律神経系

7. **中脳水道周囲灰白質（PAG）**
   - 基本的な情動反応（凍結、逃走、闘争）
   - 入力: 扁桃体、ACC
   - 出力: 脳幹運動系、エンドルフィン放出

8. **腹側線条体/側坐核（Ventral Striatum / NAc）**
   - 報酬処理、動機づけ
   - 入力: VTA（ドーパミン）、PFC、扁桃体
   - 出力: 淡蒼球、PFC
   - ドーパミン報酬予測誤差がここで統合される

9. **脳幹核群（Brainstem Nuclei）**
   - 青斑核（LC）: NE産生→覚醒・注意
   - 縫線核（Raphe）: 5-HT産生→気分調整
   - VTA: DA産生→報酬・動機
   - 入力: 視床下部、PAG
   - 出力: 広域投射

# 神経伝達物質ダイナミクス

各伝達物質は tonic（基底レベル）+ phasic（位相的バースト）で表現し、
再取り込み・分解・ホメオスタシスによる自然回帰を持つ。

| 伝達物質 | 産生源 | 主要標的 | 情動における役割 |
|---------|--------|---------|----------------|
| ドーパミン（DA） | VTA/SNc | NAc, PFC | 報酬予測誤差、動機づけ、快感 |
| セロトニン（5-HT） | 縫線核 | 広域 | 気分調整、衝動抑制、社会行動 |
| ノルエピネフリン（NE） | 青斑核 | 広域 | 覚醒、注意、闘争逃走 |
| GABA | 広域 | 局所抑制 | 不安軽減、扁桃体の抑制制御 |
| グルタミン酸（Glu） | 広域 | 局所興奮 | シナプス伝達、学習（LTP） |
| オキシトシン（OXT） | 視床下部 | 扁桃体等 | 社会的結合、信頼、脅威緩和 |
| コルチゾール（CORT） | 副腎（HPA軸） | 海馬、PFC等 | ストレス応答、記憶変調 |
| エンドルフィン（END） | PAG, 弓状核 | PAG, NAc | 疼痛調節、快感、ストレス緩和 |

# ホメオスタシスと身体性

- **HPA軸**: 視床下部CRH → 下垂体ACTH → 副腎コルチゾール → 海馬/PFC負のフィードバック
- **自律神経系**: 交感神経（闘争逃走）/ 副交感神経（休息消化）のバランス
- **内受容信号**: 心拍、呼吸、消化、筋緊張 → 島皮質で統合 → 情動体験に影響
- **エネルギー恒常性**: 疲労・覚醒サイクル

# 学習則

脳に準拠した学習メカニズム:
- **ヘブ則/STDP様**: 同時発火による結合強化（バックプロップではない）
- **報酬変調学習**: ドーパミン報酬予測誤差による強化
- **情動タグ付け記憶**: 扁桃体活性による海馬記憶の優先固定化
- **恐怖条件付け/消去**: 扁桃体-PFC回路の可塑性

# 評価基準（3レベル）

1. **神経活動レベル**: 各領域の発火パターン・神経伝達物質動態が生理学的範囲内か
2. **回路機能レベル**: 脅威→扁桃体活性化→凍結反応、報酬→DA放出→接近行動、等の因果連鎖
3. **行動レベル**: 複数の情動シナリオで一貫した適応的行動が出るか

# 技術方針

- Python + PyTorch（神経質量モデルのテンソル演算）
- Pydantic（スキーマ定義）
- FastAPI（外部インターフェース）
- pytest（各レベルの検証）
- JSONL（全領域活動・伝達物質・状態遷移の監査ログ）
- YAML（実験設定のバージョン管理）

# 既存レイヤーとの関係

既存の抽象レイヤー（valence/arousal等）は廃止せず、
**設計者が定義した近似関数**で回路活動から読み出す（将来的にはデータ駆動化が目標）。

注意: 以下は手動設計の線形結合であり、神経科学的に導出されたものではない。
- valence ← DA(NAc) - CORT - 扁桃体脅威出力（係数は任意設定）
- arousal ← NE(LC) + Glu活性
- threat_load ← 扁桃体出力 + CORT + NE
- trust ← OXT + PFC-扁桃体抑制強度
- fatigue ← 1 - (エネルギー恒常性)
