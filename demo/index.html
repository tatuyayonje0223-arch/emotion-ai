EmotionAI Demo

ヒト10情動の神経回路モデル (~821 spiking neurons) を操作し、 入力シナリオに対する脳領域活動の変化を可視化します。

⚠️ 研究用モデルです: 実脳の忠実な再現ではなく、定性的模倣。 Validation スコア: Izhikevich 35/36 MC-stable, AdEx 25/36。Baseline physiology 6/20 (既知の制限)。 詳細 on GitHub

入力シナリオ (0-1 で調整)

10情動 活性度 (emotion readout)

主要 populations 発火率 (Hz)

赤: 高活性(>15Hz) 青: 中(5-15) 灰: 低(<5)

全 population 発火率 (53 populations)
Disclaimer (科学的整合性):
このデモは 821 neuron の簡略化モデル (biological 10^11 neuron の4-5桁縮小) です。 Validation は firing rate matching ベースで、behavioral prediction 能力は未検証。 27件のパラメータ変更のうち一部は empirical calibration (target rate から逆算) です。 詳細は parameter_changes_log.md。
